Glue records: Place for improvisations

Note that the name servers for google.com are subdomains of google.com. This is fine, because there are glue records in the âAdditional Sectionâ the give the IP addresses explicitly. Without these, it would have been impossible to resolve any of google.comâs addresses (it would have got stuck on obtaining the address of e.g. ns1.google.com).

That isnât so trivial. For example, letâs look at netvision.net.ilâs name server record, as reported by the authoritative server for net.il:
